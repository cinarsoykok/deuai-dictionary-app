# DeuAI 🇩🇪🤖

AI-powered German learning mobile application built with React Native, Expo, TypeScript, and OpenAI APIs.

## ✨ Features

- 🇩🇪 German word analysis
- 📖 Simple German definitions
- 🌍 English & Turkish translations
- 🧠 AI-generated example sentences
- 🖼️ AI-generated learning images
- 💾 Save favorite words
- 🎨 Modern glassmorphism UI
- ⚡ Fast and responsive experience

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/cinarsoykok/deuai-dictionary-app.git
```

Go into the project folder:

```bash
cd deuai-dictionary-app
```

Install dependencies:

```bash
npm install
```

Create a `.env` file:

```env
EXPO_PUBLIC_OPENAI_API_KEY=your_api_key_here
```

Start the project:

```bash
npx expo start
```

---

## 🔐 Environment Variables

The project requires an OpenAI API key.

Create a `.env` file in the root directory:

```env
EXPO_PUBLIC_OPENAI_API_KEY=your_api_key_here
```
